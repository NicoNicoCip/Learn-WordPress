# Project 2: Build a Blog/Magazine Site

## Overview
Build a content-focused blog or online magazine site. This project emphasizes content organization, blog functionality, and reader experience.

**Estimated time:** 2-4 hours

---

## The Brief

**Client:** "TechPulse" — a tech news and opinion blog covering gadgets, software, and digital lifestyle.

**Requirements:**
- Blog-centric design optimized for reading
- Well-organized content categories
- Engaging homepage with featured and recent articles
- Author bio sections
- Newsletter signup

---

## Pages to Build

### 1. Home Page (Blog-Style)
**Content to include:**
- Header with site logo and navigation
- Featured/hero post section (latest post with large image)
- Recent posts grid (6-8 posts in a 2 or 3 column grid)
- Sidebar with:
  - About the blog (short text)
  - Categories list
  - Newsletter signup form
- "Load more" or pagination

**Elementor approach:** Use product shortcodes or build manually:
- Hero section with the latest post info (manually updated)
- Use `[recent_posts]` or build a custom layout

### 2. About Page
**Content to include:**
- About the publication
- The editorial team (3 writers with bios and social links)
- Contact information
- "Write for us" call-to-action

### 3. Contact Page
**Content to include:**
- Contact form (Name, Email, Subject, Message)
- "Pitch a Story" note for potential contributors
- Social media links

### 4. Category Pages (Auto-Generated)
WordPress automatically creates these. Make sure they look good by:
- Setting up categories properly
- Adding category descriptions
- Testing the archive page layout

---

## Blog Posts to Create

Create at least **8 blog posts** across these categories:

### Categories:
1. **Gadgets & Gear** (3 posts)
2. **Software & Apps** (3 posts)
3. **Digital Lifestyle** (2 posts)

### Sample Posts:

**Gadgets & Gear:**
1. "The Best Wireless Earbuds of 2026: Our Top 5 Picks"
   - 500+ words, featured image, comparison table
   - Tags: earbuds, wireless, audio, reviews

2. "Hands-On: The Latest Smartphone Camera Revolution"
   - 400+ words, multiple images
   - Tags: smartphone, camera, photography, review

3. "Smart Home Essentials: What You Actually Need"
   - 400+ words, list format with images
   - Tags: smart-home, IoT, automation

**Software & Apps:**
4. "10 Productivity Apps That Actually Changed My Workflow"
   - 600+ words, screenshots, list format
   - Tags: productivity, apps, workflow, tools

5. "AI Tools Everyone Should Know About"
   - 500+ words, categorized list
   - Tags: AI, tools, technology, machine-learning

6. "The Rise of No-Code Platforms: Build Without Coding"
   - 400+ words
   - Tags: no-code, development, platforms, tools

**Digital Lifestyle:**
7. "Digital Detox: How to Unplug Without Missing Out"
   - 400+ words, tips format
   - Tags: wellness, digital-detox, tips, lifestyle

8. "Remote Work Setup: Building the Perfect Home Office"
   - 500+ words, equipment list, images
   - Tags: remote-work, home-office, productivity, setup

### Post Requirements:
- Each post must have: **title, featured image, content (300+ words), category, 3-5 tags, excerpt**
- Use **headings (H2, H3)** to structure content
- Include **at least one image** in the body
- Set **Yoast SEO** title and meta description
- Use **internal links** (link between your posts)

## Design Specifications

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Electric Blue | `#0066ff` | Links, accents |
| Dark Navy | `#1a1a2e` | Headers, dark backgrounds |
| Light Gray | `#f4f4f5` | Background, cards |
| Dark Text | `#222222` | Body text |
| Medium Gray | `#666666` | Meta text, captions |

### Fonts
- **Headings:** Inter or Roboto (Modern, clean)
- **Body:** Source Serif Pro or Georgia (Easy on the eyes for reading)

### Photography
- Use Unsplash/Pexels for tech-related images
- Search terms: technology, laptop, smartphone, coding, workspace, gadgets
