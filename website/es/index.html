Curso WordPress

Un curso práctico para aprender a construir sitios web con WordPress y Elementor, desde principiante total hasta constructor de sitios con confianza.

Cómo Usar Este Curso

  1. Configura tu entorno local primero (Módulo 1, Lección 02)
  2. Sigue las lecciones en orden — cada una se construye sobre la anterior
  3. Haz todos los ejercicios — leer no es suficiente, necesitas practicar
  4. Completa los proyectos prácticos al final para consolidar tus habilidades

Cada lección contiene:

  • Objetivos — lo que aprenderás
  • Instrucciones — guía paso a paso
  • Ejercicios — práctica con las manos en la masa
  • Puntos Clave — resumen de conceptos importantes

Proyectos Prácticos

Proyecto 1: Sitio Web Empresarial

Construye un sitio web completo de 5 páginas para una cafetería ficticia. Tiempo estimado: 3-5 horas.

Proyecto 2: Blog / Revista

Construye un blog de tecnología con más de 8 entradas y navegación por categorías. Tiempo estimado: 2-4 horas.

Proyecto 3: Tienda E-Commerce

Construye una tienda WooCommerce completa con más de 12 productos. Tiempo estimado: 4-6 horas.

Herramientas que Necesitarás

  • Local WP (gratuito) — Entorno de desarrollo local para WordPress
  • Un navegador moderno (Chrome recomendado por sus DevTools)
  • Un editor de texto (VS Code recomendado) — para cuando necesites ver/editar archivos de código
  • Un editor de imágenes (el nivel gratuito de Canva funciona genial) — para crear imágenes para tus sitios
